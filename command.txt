
Доступные команды:

	ls - показать все папки в текущей директории.
	cd <глобальный путь до директории> или cd <локальный путь до директории> - переход в другую директорию.
	cd ..\ - для того, чтобы перейти в родительскую директорию.
	dr - доступные диски.
	mv <путь до существующей папки> <путь по которому переместить папку> - перемещение папки.
	rm <имя папки/файла> - удаление папки или файла.
	pls <путь до первого файла> <путь до второго файл> [имя файла] - конкатенция файлов.
	dev - тестовый режим для разработчиков, который создает папку develop_mode.txt с данными.
	read <имя текстового файла> - чтение текстового файла.
	create <имя файла> [кодировка] - создание файла в заданной кодировке:
		create <имя файла> utf8 - фcайл UTF8
		create <имя файла> unicode - файл Unicode
		create <имя файла> ascii - файл ASCII
	copy <имя файла, которого хотите скопировать> - копирование файла. 
	Копирование возможно только в текущей директории.
	first - переход в начальную директорию.
	clear - очистка консоли.
	exit - выход из программы.

Дополнительные команды:

	reg <элемент> - поиск файлов по заданному элементу в текущей директории. Например reg .txt
		При использовании reg, элемент вводится БЕЗ * в начале.
	regall <элемент> - поиск файлов по заданному элементу во всех директориях.
	diff <первый файл> <второй файл> - сравнение двух текствовых файлов.
	Рекомендую перейти в директорию different (cd different),
	затем прописать (diff test1.txt test2.txt).
	Также для наглядности можете сравнить  diff test1.txt test4.txt
	
Замечание:
	Совету конкатенировать файлы в одной кодировке. Удачного пира, проверяющий!
